# вњЁ Features

## Home
- **Featured Playlists** (focused on Tamil chartbusters)
- **Recently Played** & **Made For You**
- **Trending Albums** & **Popular Artists** (Anirudh Ravichander, A.R. Rahman)

## Sidebar Navigation
- **Home**, **Search**, **Your Library**
- **Dynamic Playlist list**

## Search
- **Search Bar** - live query matching against song title, artist, or album.
- **Browse Genres** - colorful card grids.

## Music Player
- **Web Audio API Synth** - plays actual chip-tune electronic synth arpeggios for each song when clicked.
- **Jumping Visualizer** - dynamic jumping bar graphs sync to playback.
- **Sliders & Scrubbing** - timeline progress bar and volume level adjustment.
- **Skipping** - Next/Previous track skipping.
- **Shuffle & Repeat**

## Playlist & Album
- **Header Banners** - with dynamic extraction gradients.
- **Song Tables** - list details (index, title, album, date added, duration, hover hearts).
- **Favorites Integration** - click heart icon to save songs, dynamically updating lists.

## Artist Page
- **Verified Artist Badge**
- **Listeners Count**
- **Popular Tracks list**
- **Artist Biography panel**

## Library
- **Tab Filters** - toggle views of playlists, albums, and artists.
